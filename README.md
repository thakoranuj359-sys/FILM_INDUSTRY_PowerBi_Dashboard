# 🎬 Films & Artists Dashboard

## Overview
An interactive Power BI project analyzing films and artist data.  
It highlights film releases, budgets, ratings, certificates, and contributions of producers and directors.

## Features
- Film trends by year  
- Budget vs rating analysis  
- Certificate distribution (U, U/A, A)  
- Producer & director performance  
- Gender representation in films  

## Tools Used
- Power BI  
- Data Cleaning & Transformation  
- Visualization Design  

## Repository Contents
- `FilmDashboard.pbix` – Power BI dashboard file  
- `Dataset/` – Film and artist data
